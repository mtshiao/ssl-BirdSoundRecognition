Bird Sound Classifier using Self-Supervised Learning

This repository contains the code and resources for our bird sound classifier, which leverages self-supervised learning (SSL) techniques for improved recognition of bird species.
Key Features:
1. Implements Audio-Masked Autoencoder (MAE) for self-supervised learning.
2. Handles data imbalances and cross-domain challenges using augmentation and open-source datasets.
3. Supports long-term monitoring of bird distribution and vocal activity in remote montane areas.
